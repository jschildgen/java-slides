Aufgabe 2: Länder-Spiel

Entwickeln Sie das folgende Spiel in Java (alles in einer Klasse):

  • Zunächst wird der Spieler aufgefordert, die Namen aller Spieler einzugeben. Das Spiel kann mit beliebig vielen Spielern gespielt werden (mind. 2).
  • Danach wird reihum jeder Spieler gefragt ein Land einzugeben.
  • Nennt ein Spieler ein Land, welches schon einmal genannt wurde, verliert der Spieler und scheidet aus.
  • Derjenige Spieler, der als einziges übrig bleibt, bekommt einen Punkt.
  • Danach kann eine neue Runde gestartet werden oder das Spiel wird beendet.
  • Am Ende soll ausgegeben werden, wie viele Punkte jeder Spieler erreicht hat.

Beispiel-Ablauf einer Runde: Achim, Berta und Carla spielen. Achim nennt Deutschand, Berta sagt Italien, Carla sagt USA, Achim sagt Frankreich, Berta Deutschland. Damit scheidet Berta aus, weil Deutschland schon einmal genannt wurde, und es geht weiter mit Carla, dann Achim, dann wieder Carla, usw. Da Achim irgendwann das bereits genannte Land Frankreich nennt, gewinnt Carla einen Punkt.
