Betrachten Sie das hier gezeigte Java-Programm, welches eine CSV-Datei aus dem Internet abruft und verarbeitet.

Die in der APIHelper-Klasse befindliche Methode httpGetList ist eine leicht abgeänderte Variante der Methode vom Übungsblatt über die Witze-App. Diese gibt nicht einfach einen String aus, sondern eine List<String>. Jedes Listenelement entspricht einer Zeile.

  • Öffnen Sie die URL zur CSV-Datei, die hier abgerufen wird, in einem Browser und betrachten Sie die Datei in einem Texteditor (nicht in Excel o. ä.)
  • Versuchen Sie die in der Main-Klasse stehenden Lambda-Ausdrücke zu verstehen und nachzuvollziehen. Beantworten Sie dazu die Verständnisfragen im Moodle (kurze Antworten; max. 20 Wörter).
  • Ergänzen Sie das Programm so, dass die Liste aller Länder ausgegeben wird und führen Sie das Programm aus. Verwenden Sie dazu keine Schleife, sondern laender.forEach
