Wir befinden uns im Finale eines großen Mau-Mau-Turniers. Es liegt eine rote 10 auf dem Tisch. Die Spielerin hat nur noch eine zufällige Karte auf der Hand. Sie zieht und zieht, bis sie endlich legen kann.

Erzeugen Sie eine rote 10 und in einer Schleife, so lange eine weitere Karte, bis diese auf die rote Zehn gelegt werden kann. Geben Sie danach "Na endlich!" auf der Konsole aus.
