Aufgabe 1: IntelliJ Edu

Wir werden in den Übungsblättern häufig das IntelliJ-Plugin namens EduTools einsetzen. Im Moodle finden Sie dann Jar-Dateien zum Download, welche Sie mit IntelliJ Edu importieren können. Daraufhin steht Ihnen ein Java-Projekt zur Verfügung, dessen Code Lücken enthält, die Sie füllen müssen. Das Plugin zeigt Ihnen die Aufgabenstellungen sowie das Feedback über Ihre Lösungen.

  1. Falls Sie IntelliJ noch nicht installiert haben, tun Sie dies nun (siehe Übungsblatt 1, Aufgabe 2). Sie können auch direkt die IntelliJ Edu Version installieren, dann entfällt Schritt 2.
  2. Installieren Sie das EduTools-Plugin in IntelliJ: File (oder wenn kein Projekt geöffnet ist: Configure) → Settings... → Plugins → Marketplace → EduTools → Install → Restart IDE
  3. Sofern Sie ein Projekt geöffnet haben, schließen Sie dieses: File → Close Project.
  4. Laden Sie das EduTools-Archiv für diese Woche aus dem Moodle herunter und importieren Sie es in IntelliJ: Learn and Teach → Import Local Course →
  5. Lösen Sie die in IntelliJ gezeigte Aufgabe.
