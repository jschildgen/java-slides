Für das Kartenspiel Mau-Mau wird eine Klasse Karte benötigt.

Wie im folgenden UML-Diagramm zu sehen, besitzt diese Klasse die Member-Variablen farbe und wert sowie die entsprechenden Getter- und Setter-Methoden für ebendiese beiden Attribute. Zusätzlich gibt es einen Konstruktor ohne Parameter und einen Konstruktor, der eine Karte mit Farbe und Wert initial belegt. Die Kartenfarbe wird als Aufzählungstyp FARBE implementiert, der die Werte ROT und SCHWARZ annehmen kann.

Entwickeln Sie die Klasse mit den im UML-Diagramm gezeigten Member-Variablen und Methoden.

In der Main-Klasse können Sie nachschauen, wie man die Klasse verwendet.

Tipp: IntelliJ kann Ihnen den Code für die Getter und Setter sowie für die Konstrukoren automatisch erzeugen!
