Aufgabe 1: javac und java

In dieser ersten Aufgabe installieren Sie das Java Development Kit JDK und kompilieren und starten Ihr erstes Java-Programm. Lösen Sie diese Aufgabe bitte ohne Verwendung einer IDE (wie zum Beispiel IntelliJ)!

  1. Prüfen Sie zunächst, ob das JDK bereits auf Ihrem PC installiert ist, indem Sie javac -version in einer Konsole (Linux/MacOS) bzw. javac.exe -version in der PowerShell (Windows) eingeben. Wird eine Version größer als 8 (bzw. 1.8) angezeigt, fahren Sie mit Aufgabenteil 3 fort.
  2. Installieren Sie das JDK der Java Standard Edition (Java SE). Sie finden es auf der Oracle-Webseite für Linux, Mac und Windows. Bei den tar.gz- bzw. zip-Dateien ist keine Installation sondern lediglich ein Entpacken notwendig. Empfehlenswert ist jedoch, dass JDK zu installieren.
  3. Erstellen Sie das Beispiel-Programm aus Vorlesungs-Kapitel 1, Folie 16 und speichern Sie es ab unter MeinProgramm.java
  4. Kompilieren Sie das Programm mittels javac und starten Sie es mit java
  5. Erweitern Sie das Programm, sodass es den Benutzer mit seinem oder ihrem Namen begrüßt. Wenn Sie das z. B. Programm mittels java MeinProgramm Peter aufrufen, soll "Hallo Peter!" ausgegeben werden. In der Methode main(String[] args) bietet Ihnen args[0]Zugriff auf diesen ersten Konsolenparameter.

Aufgabe 2: IntelliJ

In dieser Aufgabe geht es darum, die IDE IntelliJ zu installieren und dort eine Anwendung zu entwickeln.

  1. Falls Sie IntelliJ noch nicht installiert haben, downloaden Sie die Installationsdateien der kostenlosen IntelliJ IDEA Community Edition von jetbrains.com und installieren Sie das Programm.
  2. Erstellen Sie in IntelliJ ein neues Java-Projekt. Sie können das Template "Command Line App" verwenden, um direkt mit einer Klasse zu starten, die eine main-Methode beinhaltet.
  3. Probieren Sie Live-Templates aus: Geben Sie sout oder main im Code-Editor ein und drücken Sie die Tabulator-Taste.
  4. Ihr Programm soll zunächst einfach nur "Hallo!" ausgeben.
  5. Benutzen Sie eine Suchmaschine, um etwas über den Umgang mit Datum und Uhrzeit in Java zu erfahren. Ihr Programm soll nämlich die aktuelle Uhrzeit ausgeben, z. B. "Hallo, es ist 14:28 Uhr."
